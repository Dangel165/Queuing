# 🎣 소셜 미디어 피싱 공격 도구

Facebook, Twitter의 로그인 페이지를 복제한 피싱 공격 도구입니다.

## 🎯 피싱 공격 기능

### 🔍 사이트 복제
- **Facebook**: 실제 Facebook 로그인 페이지 디자인  
- **Twitter(X)**: 실제 Twitter 로그인 페이지 디자인
- 피해자가 구별할 수 없는  복제본

### 📱 QR 코드 피싱
- 가짜 로그인 페이지로 연결되는 QR 코드 생성
- 모바일 사용자를 대상으로 한 피싱 공격
- 소셜 엔지니어링과 결합하여 효과적인 공격 가능

###  실시간 수집
- 입력된 모든 아이디/비밀번호를 실시간으로 수집
- 관리자 패널에서 수집된 정보 확인
- IP 주소, 시간, 플랫폼별 분류 저장

### 🎭 위장
- 로그인 성공 시 실제 소셜 미디어 사이트로 리다이렉트
- 피해자가 공격을 눈치채지 못하도록 자연스러운 흐름
- 실제 서비스와 동일한 사용자 경험 제공

## 🚀 공격 설정 및 실행

### 방법 1: 명령행 실행
```bash
# 통합 메뉴
python 큐싱(Qshing).py.py

# 피싱 서버만
python 큐싱(Qshing).py server

# QR 피싱 도구만
python 큐싱(Qshing).py qr
```

## 🌐 피싱 사이트 접속

### 가짜 로그인 페이지
- **메인 페이지**: http://localhost:5000
- **가짜 Facebook**: http://localhost:5000/facebook
- **가짜 Twitter**: http://localhost:5000/twitter
- **공격자 패널**: http://localhost:5000/admin

### QR 코드 피싱 생성
피싱 QR 코드 생성:
```bash
python 큐싱(Qshing).py qr
```

## 🎣 피싱 공격 방법

### 자격증명 수집
- **모든 입력된 정보가 자동으로 수집됩니다**
- 아이디, 비밀번호, IP 주소, 접속 시간 기록
- 피해자는 정보가 탈취되었다는 것을 알 수 없음

### 공격 시나리오 예시
1. **이메일 피싱**: 가짜 보안 알림 이메일에 피싱 링크 포함
2. **QR 코드 피싱**: 공공장소에 가짜 QR 코드 배치
3. **소셜 엔지니어링**: 긴급한 로그인이 필요하다고 속여서 링크 클릭 유도

## 📱 QR 코드 피싱 공격

### 피싱 QR 생성
```
피싱 QR 코드를 생성할 플랫폼을 선택하세요:
1. 👥 Facebook 피싱
2. 🐦 Twitter 피싱  
3. 🌟 전체 피싱 QR 생성
0. QR 생성기만 종료
9. 전체 프로그램 종료
```

### 생성되는 피싱 파일
- `facebook_qr_20240201_143023.png` (Facebook 피싱 QR)
- `twitter_qr_20240201_143024.png` (Twitter 피싱 QR)

## �️ 공격자 대시보드

### 실시간 피해자 모니터링
- 플랫폼별 피싱 성공률
- 수집된 자격증명 통계
- 총 피해자 수

### 수집된 정보 관리
- 시간별 피싱 공격 기록
- 피해자 자격증명 정보 (아이디/비밀번호)
- IP 주소 추적 및 지역 분석
- 플랫폼/성공률별 필터링

### 데이터 추출
- JSON 형태로 수집된 정보 저장
- `login_logs.json` 파일에 모든 피해자 정보 기록
- 추후 분석을 위한 데이터 보관

## 🛠️ 공격 도구 스택

- **Backend**: Python Flask (경량화된 피싱 서버)
- **Frontend**: HTML5, CSS3, JavaScript (사이트 복제)
- **QR Code**: qrcode 라이브러리 (피싱 QR 생성)
- **UI Framework**: 순수 CSS (각 플랫폼  모방)
